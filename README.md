# Latih-Minda
Platform permainan digital untuk kanak-kanak berumur 2–5 tahun yang membantu pembelajaran asas seperti huruf, nombor, menulis, warna, haiwan dan bentuk. Bukan menggantikan ibu bapa, tetapi menjadikan masa skrin sebagai masa belajar bersama agar ibu bapa lebih mengetahui dan terlibat dalam pembelajaran anak.
